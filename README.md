A Machine Learning-based application that predicts Bitcoin prices using historical data and AI/ML techniques. The project uses Logistic Regression and provides predictions through a web interface and Streamlit app.

---

## 🚀 Features
-  Predict Bitcoin prices using trained ML model
-  Model built with Logistic Regression
-  Web interface using HTML, CSS (Flask)
-  Interactive UI using Streamlit
-  Uses real Bitcoin dataset for training

---

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- Flask (HTML, CSS)
- Jupyter Notebook

## 🧠 How It Works

1. Data preprocessing on historical Bitcoin dataset  
2. Feature selection and training using Logistic Regression  
3. Model saved using joblib  
4. User inputs values via UI  
5. Model predicts price trend (increase/decrease)
